# Kenney 3D models (CC0)

Models in this folder are from [Kenney](https://kenney.nl/assets) (CC0 1.0).

| File | Source pack |
|------|-------------|
| `arcade-machine.glb`, `pinball.glb`, `air-hockey.glb` | Mini Arcade |
| `laptop.glb`, `computerScreen.glb`, `computerKeyboard.glb`, `computerMouse.glb`, `lampRoundTable.glb`, `speaker.glb`, `speakerSmall.glb` | Furniture Kit |
| `pallet.glb`, `planks.glb`, `detail-cables-type-a.glb` | Retro Urban Kit |

**Textures:** Several GLBs reference external atlases in `Textures/` (not embedded in the binary). Required files:

- `colormap.png` — Mini Arcade props
- `planks.png`, `signs.png` — Retro Urban props
