# JJ Heffernan — Dev Hub

GitHub Pages dev hub for [jjheffernan.github.io](https://jjheffernan.github.io) — a 3D index of repos, organizations, and public sites.

**Portfolio (the polished personal site):** [jjheffernan.netlify.app](https://jjheffernan.netlify.app/) · [cosmic-themes-starter](https://github.com/jjheffernan/cosmic-themes-starter)

## Stack

- React + TypeScript + Vite + Tailwind CSS
- React Three Fiber, drei, postprocessing
- GitHub Pages via GitHub Actions

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Pushes to `main` deploy automatically through `.github/workflows/deploy.yml`.
